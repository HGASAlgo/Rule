### 윤영

### 공통 규칙

- 중요한 일정이 있을 경우 일주일 전에 말하면 면제
- 알고리즘 풀이 주석 최소 3줄 적기
- 벌금은 2의 지수승으로 증가
  - Ex) 한 번 - 이천원 / 두번 - 사천원 / 세번 - 팔천원 / 네번 - 만육천원
- 매달 초에 문제 개수 변경 가능
